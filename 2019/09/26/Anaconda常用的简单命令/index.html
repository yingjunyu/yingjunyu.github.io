Anaconda常用的简单命令

conda info -e 查看当前虚拟环境

conda create python36 创建了python36的虚拟环境

activate python36 激活python36的虚拟环境

conda list 查看已经安装的包

conda install package_name 安装一些包，其中package_name为包的名字

conda remove package_name 删除一些包，其中package_name为包的名字

conda update package_name 更新一些包，其中package_name为包的名字

conda search package_name 查找一些包，其中package_name为包的名字

conda remove python36 删除虚拟环境
